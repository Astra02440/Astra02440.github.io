World Wide Web

Le World Wide Web est le premier site web de internet créé par TIM Bernes-Lee un chercheur britannique, créé au CERN (Organisation européennepour la recherche nucléaire en 1989 .Le World Wide Web qui est mieux connu sous le nom de «WWW». C’est un gigantesque réseau de nœuds qui nous permet de visualiser tous types de contenu, texte, vidéo ou image sur notre appareil, qu’il s’agisse d’un téléphone mobile ou d’un ordinateur.
